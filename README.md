# 🚀 Telegram Message Value Classifier API

یک سرویس API بسیار سبک، فوق‌العاده سریع و کاملاً قابل تنظیم برای تشخیص ارزشمند بودن پیام‌های تلگرام. این سیستم بدون استفاده از هیچ دیتابیس یا سرویس هوش مصنوعی (LLM/API) کار می‌کند و با تکیه بر منطق قانون‌محور (Rule-Based Matching)، عبارت‌های منظم (Regex)، وزن‌دهی کلمات کلیدی (Keyword Scoring) و فیلترهای منفی قدرتمند، در کمتر از ۱۰ میلی‌ثانیه پیام‌ها را دسته‌بندی و تحلیل می‌کند.

---

## 🎯 اهداف پروژه

- **ورودی API:** متن پیام تلگرام همراه با اطلاعات اختیاری فرستنده و گروه.
- **خروجی API:** تعیین این که آیا پیام ارزش فوروارد شدن و پیگیری دارد یا خیر.
- **دسته‌بندی‌های هدف:**
  1. `hiring`: استخدام یا جذب نیرو
  2. `project_request`: درخواست انجام پروژه (فریلنسری)
  3. `collaboration`: همکاری کاری یا شراکت / سرمایه‌گذاری
  4. `irrelevant`: نامرتبط یا اسپم


## ⚡ منطق پردازش (Processing Pipeline)

1. **نرمال‌سازی (Normalization):** تبدیل کاراکترهای عربی به فارسی (ي ➔ ی، ك ➔ ک)، بهینه‌سازی نیم‌فاصله‌ها، تبدیل حروف انگلیسی به lowercase، حذف فاصله‌های تکراری و حذف علائم تزئینی بدون خراب کردن فارسی.
2. **رد سریع (Fast Reject):** پیام‌هایی که کمتر از ۲۰ کاراکتر هستند، فقط شامل لینک هستند، فقط شامل ایموجی هستند، فقط عدد یا فرمول‌های کدی هستند، بیشتر از ۶۰٪ پیام هشتگ/منشن/لینک هستند و یا شامل کلمات شدیداً بلاک‌شده (کازینو، شرط‌بندی، ترید، فارکس، بت و...) هستند، بلافاصله در کسری از میلی‌ثانیه رد می‌شوند.
3. **امتیازدهی مثبت (Positive Scoring):** کلمات کلیدی هر سه دسته هدف جستجو شده و امتیاز پایه می‌گیرند. در صورت همخوانی کلمات شاخص، حضور عبارات تعدیل‌کننده (مانند دورکاری، حقوق، سرمایه، بودجه و...) امتیاز اضافی به همراه خواهد داشت.
4. **امتیازدهی منفی (Negative Scoring):** در صورت وجود سیگنال‌های اسپم، تبلیغات عمومی، ممبرگیری، فروش کالا یا پکیج‌های آموزشی، امتیازهای جریمه منفی اعمال می‌شود.
5. **تصمیم‌گیری نهایی (Decision Engine):** امتیاز نهایی از تفاضل امتیاز مثبت دسته غالب و جریمه منفی به دست می‌آید. اولویت سیستم بر روی **Precision (دقت بالا)** متمرکز است تا پیام‌های نامربوط فیلتر شوند.

## 💻 نصب و راه‌اندازی (Setup & Installation)

### پیش‌نیازها
- Python 3.8 یا بالاتر

### مراحل نصب
۱. وارد پوشه پروژه شوید:
```bash
cd "message-value-api"
```

۲. یک محیط مجازی (Virtual Environment) بسازید و فعال کنید:
```bash
python -m venv venv
source venv/bin/activate  # در ویندوز: venv\Scripts\activate
```

۳. کتابخانه‌های مورد نیاز را نصب کنید:
```bash
pip install -r requirements.txt
```

۴. سرور توسعه را اجرا کنید:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

اکنون وب‌سرویس روی آدرس `http://localhost:8000` در دسترس است. همچنین مستندات تعاملی Swagger روی `http://localhost:8000/docs` فعال خواهد بود.

## 📡 اندپوینت‌های API (Endpoints)

### ۱. تست سلامت سرویس
- **روش (Method):** `GET`
- **آدرس (URL):** `/health`
- **پاسخ نمونه:**
```json
{
  "status": "ok",
  "rules_loaded": true
}
```

### ۲. دسته‌بندی و سنجش پیام
- **روش (Method):** `POST`
- **آدرس (URL):** `/classify`
- **نمونه درخواست (Request Body):**
```json
{
  "text": "به یک نفر برنامه نویس پایتون و جنگو برای یک پروژه جذاب نیازمندیم. کار به صورت کاملا دورکاری و تمام وقت هست و حقوق بسیار خوبی پرداخت میشه. ارسال رزومه به آیدی زیر.",
  "chat_title": "گروه توسعه‌دهندگان ایران",
  "sender_name": "Ali_Dev"
}
```
- **نمونه پاسخ (Response Body):**
```json
{
  "valuable": true,
  "category": "hiring",
  "score": 11.0,
  "confidence": 0.89,
  "reasons": [
    "Matched core 'hiring' keyword(s): نیازمند, python, django, رزومه",
    "Matched 'hiring' modifier(s): حقوق, تمام وقت, دورکاری, ارسال رزومه",
    "High final score (11.0) indicating highly valuable content."
  ],
  "matched_keywords": [
    "django",
    "python",
    "ارسال رزومه",
    "تمام وقت",
    "حقوق",
    "دورکاری",
    "رزومه",
    "نیازمند"
  ],
  "negative_reasons": []
}
```

### ۳. بارگذاری مجدد و آنی قوانین (Hot-Reload Rules)
با ویرایش فایل `rules.yaml` نیازی به ری‌استارت کردن وب‌سرویس نیست. کافیست این اندپوینت را صدا بزنید تا قوانین در لحظه به‌روزرسانی شوند.
- **روش (Method):** `POST`
- **آدرس (URL):** `/reload-rules`
- **نمونه پاسخ:**
```json
{
  "status": "ok",
  "message": "Rules reloaded successfully.",
  "rules_count": {
    "positive_categories": [
      "hiring",
      "project_request",
      "collaboration"
    ],
    "spam_keywords_count": 11
  }
}
```
